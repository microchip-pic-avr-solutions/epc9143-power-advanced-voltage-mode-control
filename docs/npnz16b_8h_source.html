138  * *************************************************************************************************
139  * Summary:
140  * Filter Coefficient Arrays, Number Format Handling and Input/Output History Parameters
141  *
142  * Description:
143  * The NPNZ_PORTS_t data object holds a list of nested NPNZ_PORT_t data objects, each
144  * defining an individual controller input or output port. The NPNZ16b_t data objects defines
145  * up to two input and two output ports of type struct NPNZ_PORT_t and one additional
146  * pointer to an external, user-defined 16-bit reference source variable. Each port of type
147  * struct NPNZ_PORT_t declares its individual source/target memory address, normalization
148  * scaler and offset:
149  *
150  * - Primary Source: common feedback input object
151  * - Alternate Source: additional, alternate feedback input object (optional)
152  * - Primary Target: common control output target object
153  * - Alternate Target: additional, alternate control output target object (optional)
154  * - Control Reference: pointer to external 16-bit reference source variable
155  *
156  * *************************************************************************************************/
157
