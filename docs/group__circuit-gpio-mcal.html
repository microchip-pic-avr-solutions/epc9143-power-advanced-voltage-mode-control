Detailed Description

Global abstraction labels of special circuit signal device pin assignments.

Description: This section is used to define labels of hardware specific signals, which are directly assigned to specific device pins. These labels will be used throughout the code and remain unchanged even if this firmware is migrated to another device or pin-out changes between hardware revisions.
