Detailed Description

Global defines for PWM settings of DV330101.

Description: This section defines fundamental PWM settings required for the interleaved buck converter of the EPC9143 300W 16th brick power module reference design. These settings are determined by hardware.

Pre-compiler macros are used to convert physical values into binary (integer) numbers to be written to Special Function Registers (SFR).
